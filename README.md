# MiniRPG

## Descrição do projeto

MiniRPG é um projeto pessoal. Se trata de um jogo de RPG feito em Java com a finalidade de praticar lógica de programação, onde estarei inserindo novas funcionalidades de acordo com as ideias que eu tiver ao longo do tempo.

## Tecnologias

- Java 25
- IntelliJ
- Maven
